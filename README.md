# AI DISCORD BOT

This project is an AI bot for Puff The Dragon discord server. 

The objective of the bot is to offer on demande and personal Q&A to discord members in private threads.

### Getting started
1. Clone repository

2. Install libraries:
```cmd
npm install
```
3. Start the bot:
```cmd
npm run start
```

### Stack
Frontend => Discord

Backend => Node.js, OpenAI

Database => PostgreSQL
